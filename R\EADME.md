# HoldInfo Clone

HoldInfo App fetches data from API and gives updated data.
This project is for an Internship program made in company demand.


## Features

- App uses axios library to fetch data from API and store it in DB
- Responsive FrontEnd


## Run Locally

Clone the project

```bash
  git clone https://github.com/Immortals430/HoldInfo-Clone.git
```

Go to the project directory

```bash
  cd HoldInfo-Clone
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  node index.js
```
